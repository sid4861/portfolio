What are functions ?

The most simple and relatable definition would be, a function is just a group of code which you can give a name (also known as binding) and use that name to invoke this functionality anytime.

Why would anyone want to use functions and not just write programs line after line ?
Below are some of this benefits and uses of functions.

  • Functions help to structure your programs. As people write software, the amount of code and the documentation becomes huge. In such a situation it important to divide the program into logical unit called functions, which eventually make the code much more readable and find bugs later on.
  • Functions help to reduce repetition. If you come across a use case or task which would be required in the software at different locations and situations, you can isolate the such code into a function, and instead of writing the whole code again, just invoke the function. It is a huge time saver, also reduces the number of bugs that would be introduced when writing the whole code manually.

How do you define functions in javascript ?

Before discussing ways to define a function, just remember that functions can have a set of parameters (a function without any paramaters is also valid) and a body, the code that the function executes wrapped in curly braces

There are 3 ways of doing so

  1. Create a function and assign it to a variable

  3. Arrow functions

    4. it uses an arrow (=>) made up of an equal sign and a greater-than character and can be used to create concise functions, showm below -

    const square = x => x*x;

    Carefully observing the above code snippet, one can interpret that if the functions accepts only 1 parameter, it need not be enclosed in paranthesis, also if the body contains just one statement, we can omit the return key word and write the statement in the same line.
